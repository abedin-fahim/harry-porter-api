Harry Porter API

New Characters are being added everyday, but hope you find it useful.

  • https://abedin-harry-porter-api.herokuapp.com/characters - view all the characters
  • https://abedin-harry-porter-api.herokuapp.com/characters/students - Get all the Hogwarts students
  • https://abedin-harry-porter-api.herokuapp.com//characters/staff - Get all Hogwarts staff
  • https://abedin-harry-porter-api.herokuapp.com//characters/house/:house - Get all the characters from a certain house, e.g. /slytherin
